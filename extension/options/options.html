关于 / About

🤖

AI Content Checker

版本 / Version: 1.0.0

一款Chrome浏览器扩展，用于检测网页中的AI生成内容（文本、图片）。 支持黑白名单管理、自定义扫描范围、URL历史记录，以及第三方AI检测API集成。

A Chrome browser extension for detecting AI-generated content (text, images) on web pages. Features whitelist/blacklist management, customizable scan scope, URL history, and third-party AI detection API integration.

功能特性 / Features

  • 🔍 检测文本和图片中的AI生成内容 / Detect AI content in text and images
  • ✅ 白名单：标记可信网站，跳过检测 / Whitelist: Mark trusted sites to skip detection
  • ⛔ 黑名单：标记已知AI来源 / Blacklist: Mark known AI sources
  • 🖱️ 自定义扫描范围：选择页面上的特定区域 / Custom scope: Select specific areas on page
  • 📋 URL历史记录：追踪已检测页面的结果 / URL history: Track scan results for pages
  • 🔑 支持第三方API：GPTZero、Originality.ai、Hive / Third-party APIs: GPTZero, Originality.ai, Hive
  • 🌐 支持中英双语界面 / Bilingual Chinese/English interface

检测方法 / Detection Methods

本插件使用以下方法检测AI生成内容：

  • 启发式算法 / Heuristics: 分析句子均匀性、词汇多样性、AI常用短语、段落结构等 / Analyzes sentence uniformity, lexical diversity, AI phrases, paragraph structure
  • URL分析 / URL Analysis: 检测已知AI图像服务的URL模式 / Detects URL patterns from known AI image services
  • 元数据分析 / Metadata: 检查图片尺寸、alt文本等AI生成特征 / Checks image dimensions, alt text for AI-typical characteristics
  • API集成 / API Integration: 可接入专业AI检测服务获得更高准确率 / Integrates with professional services for higher accuracy

数据存储说明 / Data Storage

所有数据（白名单、黑名单、历史记录、API密钥）均存储在浏览器本地，不会上传到任何服务器。 API密钥仅在您配置了第三方服务时才会使用，且仅发送到对应的官方API端点。

All data (whitelist, blacklist, history, API keys) is stored locally in the browser and never uploaded to any server. API keys are only used when you configure third-party services and are only sent to the respective official API endpoints.
